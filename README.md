# AI-ErrorBook
中山大学软件工程课程期末大作业  
本项目使用了豆包的图像识别以及DeepSeek，对上传的错题分别进行OCR与错因分析，解答等；  
# 本人负责部分  
deepseek与豆包的api调用，ai功能的实现，与数据库的对接  
# 错题集与AI结合的功能  
1.识别题目，错误答案，解答，分析错因等  
2.根据近期上传的错题总结用户的薄弱知识点并推荐习题进行练习  
3.根据用户的习题上传情况制定学习计划，包括需要强化的知识点，每日练习的题量等  
# 项目的不足之处
1.无法对带图像的题目，例如数学几何题，物理情景题等的图像进行分析  
2.题目答案都是AI给出的，可能存在纰漏  
# 后续改进方案
引进图像识别相关的AI，规范内容的输出等
